Nxeed's Carousel JQuery Plugin

Простой и гибкий анимационный слайдер-карусель.

Поддерживает прокрутку по осям X и Y, темы, mousewheel, event drag, easing, двустороннюю и постраничную навигации.

Подключение

<link rel="stylesheet" href="/lib/jquery.nbc/css/jquery.nbc.css" />

<!-- Если используем слайдер баннеров -->
<link rel="stylesheet" href="/lib/jquery.nbc/themes/banners/css/theme-banners.css" />
<!-- Если используем слайдер новостей -->
<link rel="stylesheet" href="/lib/jquery.nbc/themes/tape/css/theme-tape.css" />
<!-- Если используем слайдер фотогаллереи -->
<link rel="stylesheet" href="/lib/jquery.nbc/themes/gallery/css/theme-gallery.css" />

<script type="text/javascript" src="/lib/jquery.nbc/js/jquery-1.11.1.min.js"></script>

<!-- Если используем easing -->
<script type="text/javascript" src="/lib/jquery.nbc/js/jquery.easing.1.3.min.js"></script>
<!-- Если используем draggable -->
<script type="text/javascript" src="/lib/jquery.nbc/js/jquery.event.drag.min.js"></script>
<!-- Если используем mousewheel -->
<script type="text/javascript" src="/lib/jquery.nbc/js/jquery.mousewheel.min.js"></script>

<script type="text/javascript" src="/lib/jquery.nbc/js/jquery.nbc.js"></script>

Настройки плагина

Название Описание По умолчанию Принимаемые значения
slideWidth Ширина слайда 'auto' integer — ширина в пикселах, 'auto' — определяется автоматически по первому слайду
slideHeight Высота слайда 'auto' integer — высота в пикселах, 'auto' — определяется автоматически по первому слайду
margin Расстояние между слайдами 8 integer — отступ в пикселах
items Количество отображаемых слайдов 4 integer — количество слайдов
easing Анимация пролистывания 'linear' string — название(см. http://easings.net/ru)
animationSpeed Скорость перехода к следующему слайду 400 integer — время в милисекундах
draggable Пролистывание тачем или жестами false boolean — использовать, если true
mousewheel Пролистывание колесиком мыши false boolean — использовать, если true
autoplay Автоматическое пролистывание, если курсор не наведен 5000 false — не использовать, integer — интервал в милисекундах
vertical Пролистывание по оси Y false boolean — использовать, если true
slider Двусторонняя навигация {next: 'Next', prev: 'Prev'} false — не использовать, object{next: 'Текст кнопки перехода к следующему слайду', prev: 'Текст кнопки перехода к предидущему слайду'}
pagination Постраничная навигация false boolean — использовать, если true

            mousewheel: true,
            pagination: true,
            slideWidth: 160,
            slideHeight: 66

    });
</script>
